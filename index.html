Live Block Mode

For immediate, high-priority, or emergency work that requires instant approval and track occupation.

  • Real-time track isolation
  • Instant communication with Cabin
  • Immediate field commencement

Rolling Block Mode

Allows staff to initiate requirements for future dates, facilitating a structured forwarding and sanctioning workflow.

  • Initiate future date requirements
  • Multi-level forwarding workflow
  • Advanced Officer sanctioning

Unified User Modules

Adapting to both live exigencies and rolling plan management.

Zone Admin Dashboard

  • Manage user accounts and role-based permissions.
  • Configure regional track nodes and station data.
  • Monitor system-wide Rolling vs. Live block metrics.

Officer Dashboard

  • Sanction Rolling Blocks: Approve future maintenance plans and requirements.
  • Live Control: Review and approve real-time block requests.
  • Analytics: Analyze progress reports to improve productivity.
  • Coordinate multi-station complex operations.

Cabin Holders Interface

  • Acknowledge approvals and confirm field readiness.
  • Rolling Visibility: Track upcoming sanctioned blocks for the section.
  • Provide extensions or reject with appropriate reasoning.
  • Communicate block closure and normalization.

Field View (Supervisor)

  • Initiate Rolling Block: Log future date requirements for sanctioning.
  • Initiate Live Block: Request immediate track occupation.
  • Mobile-optimized status tracking and verification.
  • Secure offline support for field operations.
